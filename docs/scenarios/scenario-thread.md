# Scenario Thread

## Scenario Title
Improving Student Satisfaction Without Increasing Costs

## Starter Question (Required)
**How do we improve professor evaluations, while avoiding an increase in cost per course?**

# Follow-Up Questions (Required)

1. What factors are associated with high professor evaluation scores?
2. Which courses have above-average evaluation scores?
3. Are high-evaluation courses also high-cost courses?
4. Do humanities courses show higher satisfaction than STEM courses?
5. How does student type (Youth vs Adult) affect evaluation trends?
6. Do professors with higher evaluations teach specific majors?
7. Which courses show stable or increasing enrollment over time?
8. What is the relationship between cost per course and evaluation score?
9. Which courses would benefit from increasing class sizes?
10. Would increasing class sizes affect student satisfaction?

# Final Recommendation
**Increase class sizes in high-performing humanities courses that have:**

- High evaluation scores  
- Moderate or below-average costs  
- Stable or increasing enrollment  

This optimizes student satisfaction without increasing cost per course.
