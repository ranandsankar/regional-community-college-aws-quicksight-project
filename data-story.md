Regional Community College aims to improve student satisfaction‚Äîmeasured through **professor evaluation scores**‚Äîwhile maintaining its commitment to **affordable education**.

**How can the college improve professor evaluations without increasing cost per course?**

### 3. **Cost Efficiency**

## üß≠ Recommendations
To implement this strategy effectively:

### **1. Identify Target Humanities Courses**
Use QuickSight filters to select courses that meet all criteria:
- Above-average evaluation scores  
- Median or below-median cost per course  
- Stable or increasing student demand  

### **2. Pilot Expanding Class Sizes**
- Add a reasonable number of seats to selected courses.  
- Monitor satisfaction scores and cost impacts closely.

### **3. Support Instructional Quality**
- Provide teaching assistants or peer leaders if class sizes increase.
- Offer faculty support for scaling their courses effectively.

### **4. Use Dashboards for Ongoing Monitoring**
Track:
- EvaluationScore trends  
- CostPerCourse shifts  
- Changes in student major distribution  
- Student Type impacts on satisfaction

## ‚ö†Ô∏èRisks, Assumptions & Next Steps

### **Assumptions**
- Increasing class size does not immediately lower course quality.  
- Cost per course remains stable within reasonable enrollment increases.  
- Student satisfaction is influenced by instruction and course design more than external factors.

### **Risks**
- Class size too large = diminished engagement  
- Humanities faculty may need additional support  
- Demand could shift unexpectedly year-to-year  

### **Next Steps**
- Run a **controlled pilot** for 1‚Äì2 terms.  
- Analyze satisfaction changes before scaling further.  
- Use Scenarios and Q&A to ask deeper questions like:  
  - *Do youth or adult students respond differently to larger classes?*  
  - *Do certain professors scale better than others?*  
  - *Which courses show the strongest correlation between class size and satisfaction?*

- Leverage existing student demand efficiently
